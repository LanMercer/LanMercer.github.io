如何搭建自己的 Hexo 博客

如果你也想拥有一个像我这样的个人博客，这篇文章会手把手教你从零开始搭建一个 Hexo 博客。

什么是 Hexo？

Hexo 是一个快速、简洁且高效的静态博客框架，使用 Markdown 写作，一键生成网页。它的优点是：

  • 免费 - 完全开源，不需要服务器费用
  • 简单 - 只需要会写 Markdown 就能发布文章
  • 美观 - 有大量免费主题可以选择
  • 可定制 - 支持自定义主题和插件
